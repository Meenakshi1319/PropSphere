# PropSphere – Real Estate Listings Web Application

## Overview
PropSphere is a modern real estate listings web application designed for the Indian market.  
It focuses on delivering a premium user experience for discovering, browsing, and exploring luxury properties across major Indian cities.

This repository currently contains the **frontend implementation** of the project.  
Backend services and database integration are planned for future updates.

## Tech Stack
- React 18
- Vite
- Tailwind CSS
- React Router DOM
- Framer Motion
- Local JSON data

## Features
- Modern, responsive UI
- Live search and filtering of properties
- Property listing and detailed property pages
- Agent contact forms
- Property submission form
- Smooth animations and transitions
- Clean, component-based architecture

## Project Structure

src/
├── components/

## How to Run the Project
1. Install dependencies: npm install
2. Start the development server: npm run dev
3. Open in browser: http://localhost:5173/
